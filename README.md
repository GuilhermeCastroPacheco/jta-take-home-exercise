# jta-take-home-exercise
Full stack application that consumes external data and presents it in clear and useful way



Configurações
1. Active created environment:
    1. cd backend
    2. venv\Scripts\activate
2. Testar backend:
    1. cd backend
    2. uvicorn main:app --reload
3. Testar frontend
    1. cd frontend
    2. npm run dev
